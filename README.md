# micro uptime
Micro uptime is inspired by both the popular tool Uptime Kuma but also my own experience renting a vps with minimal reasources. I wanted to have a similar uptime solution which would notify me when my homelab would go down but I didn't want to waste resources on running a docker container for Uptime Kuma, even though it is pretty light.

I used this reasoning as justification to learn a bit of Golang and broaden my basic programming skills to things outside of programming just for my university course.
